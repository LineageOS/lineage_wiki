## Model Identification

Before proceeding, please identify which type of h870 build you need:
* `h870` builds are only for devices that were officially bootloader unlocked on LG's web-portal before it shut down.
* `h870d` builds are only for h870 devices that utilized the [DirtySanta](https://xdaforums.com/t/guide-lg-g6-bootloader-unlock-2025.4772845/) unlock method.
